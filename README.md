# 🚀 RetailIQ.AI

## AI Powered Retail Decision Intelligence Platform

Predict prices • Optimize inventory • Maximize revenue using Artificial Intelligence


## 📌 Overview

RetailIQ.AI is an AI-powered retail intelligence platform designed to help sellers make smarter business decisions.

The platform analyzes product information, market trends, pricing patterns, and inventory conditions to provide intelligent recommendations for better profitability.


## 🎯 Problem Statement

Small and medium retailers face many challenges:

- Manual pricing decisions
- Difficulty understanding customer demand
- Changing competitor prices
- Inventory shortages and overstock problems
- Lack of data-driven business insights

Retailers need an intelligent system that can analyze market conditions and recommend better decisions.


## 💡 Solution

RetailIQ.AI provides an AI-driven decision support platform that helps retailers:

- Predict optimal product prices
- Analyze demand trends
- Monitor inventory
- Generate business insights
- Improve revenue opportunities

## ⚙️ Installation and Setup


### Clone Repository

```bash
git clone https://github.com/divyakunasani/RetailIQ-AI.git
```


### Frontend Setup

```bash
cd client

npm install

npm run dev
```


### Backend Setup

```bash
cd server

npm install

npm run dev
```
